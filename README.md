# LockedIn

## Description
A gamified to-do list app that tracks your consistency and focus.

## Features
✅ Add, edit, and delete daily tasks

📅 Monthly overview with green squares for completed days

🔥 Streak tracking with achievement badges

📊 Stats like “best streak” and “improvement from last month”

🛠 Built with DRF backend & ReactJS frontend

Achievement levels like ⭐ Star, 🌟 Superstar, 🏆 Champion, and 👑 Icon.
